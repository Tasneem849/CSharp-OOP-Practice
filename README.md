# CSharp OOP Practice

A collection of C# object-oriented programming practice projects covering the most important OOP concepts, including classes, objects, encapsulation, inheritance, polymorphism, abstraction, interfaces, and SOLID principles.

This repository is part of my learning journey as a Junior C#/.NET Developer. It contains practical console applications that demonstrate how OOP concepts can be used to solve programming problems.

---

## Concepts Covered

- Classes and Objects
- Constructors
- Encapsulation
- Properties and Fields
- Inheritance
- Polymorphism
- Method Overriding
- Abstraction
- Interfaces
- Composition
- SOLID Principles
- Clean Code Basics
